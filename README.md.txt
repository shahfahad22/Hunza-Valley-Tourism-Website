# Hunza Valley Tourism Website 🌄

This is a fully responsive tourism website for Hunza Valley built using HTML, CSS, and JavaScript.

## 🌟 Features:
- Sticky + Auto-Hide Navbar
- Hero background slideshow
- Festival & Hotel Information Pages
- Gallery with Modal Image Preview
- Contact Form
- Mobile Responsive Layout

## 🔧 Technologies Used:
- HTML5
- CSS3
- JavaScript (Vanilla)
- AOS Library for Animations

## 📸 Preview:
Live Preview: https://fancy-gnome-ed5bd0.netlify.app/

## 📁 Project Structure:
- `index.html` – Homepage
- `html/` – Other pages like gallery, hotels, festivals, contact
- `CSS/style.css` – Website Styling
- `JS/script.js` – All JavaScript logic
- `Images/` – All site images
